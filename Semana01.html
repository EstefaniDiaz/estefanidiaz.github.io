¿Que temas aprendió?

En la primera semana de clases, se exploraron los principios esenciales para el aprendizaje de tecnología web. Esto incluyó la revisión del plan de estudios, que detallaba el contenido a cubrir durante el semestre. A través de este plan, se clarificaron los objetivos educativos, delineando las habilidades y conocimientos que se espera que alcancemos al final del curso. Además, se presentaron los diversos temas que se abordarían, proporcionando una visión general de los conceptos y áreas clave que se explorarán a lo largo del semestre académico.

Definiciones

Estos temas nos permitieron entender los principios básicos del funcionamiento de la web. Aprendimos acerca del flujo de información entre un cliente y un servidor, comprendiendo el proceso de solicitud y respuesta de datos. También investigamos cómo se muestra el contenido en un navegador, lo que implica interpretar y mostrar elementos como texto e imágenes.

Estrategias de aprendizaje y tecnologías de desarrollo

La selección adecuada de estrategias de aprendizaje y tecnologías de desarrollo puede abrir escenarios académicos en los que docentes y estudiantes se apoyen en el pensamiento crítico y participen activamente en el proceso de enseñanza-aprendizaje, lo que contribuye a elevar la calidad de la educación.

Como funciona la web y que aplicaciones de internet existen

La web es un sistema de información global que permite acceder y compartir contenido a través de Internet. Funciona mediante la interacción entre servidores y clientes, donde los servidores almacenan y procesan la información, y los clientes, como los navegadores web, la solicitan y la presentan a los usuarios.
Algunas de las aplicaciones de internet más comunes son: Correo electrónico, Redes sociales, Buscadores, Comercio electrónico y Streaming.

Tecnologías web

Tecnologías web se refiere a las herramientas y lenguajes utilizados en el desarrollo de aplicaciones y sitios web. Estas tecnologías se dividen en dos categorías principales: frontend y backend. Las tecnologías frontend se centran en la parte visible de un sitio web, mientras que las tecnologías backend se ocupan de la lógica y el procesamiento de la información en el servidor.
Algunas de las tecnologías web más populares incluyen: HTML, CSS, JavaScript, PHP, Ruby, Java, Python.
Además de estos lenguajes, existen frameworks que ayudan a acelerar el desarrollo web y ofrecen soluciones a problemas comunes. Algunos ejemplos de frameworks populares son: Symfony, Laravel, Phalcon, Django, Ruby on Rails, .NET Framework.

Diseño de experiencia de interfaz

El Diseño de Experiencia de Interfaz (UX/UI) se refiere a la creación de productos digitales que sean fáciles de usar, intuitivos y agradables para los usuarios finales. Este enfoque se centra en la experiencia general del usuario, incluyendo sus percepciones, emociones y respuestas al producto, sistema o servicio de una empresa. Se compone de dos aspectos principales:

  • Diseño de Experiencia de Usuario (UX): Se enfoca en el recorrido del usuario en una aplicación o proceso, y en la forma y función general de un producto o tecnología. El objetivo del UX es garantizar que la navegación y acciones que realiza el usuario tengan coherencia y sean intuitivas
  • Diseño de Interfaz de Usuario (UI): Se centra en cómo se ve y funciona el exterior de un producto, es decir, en el aspecto visual de la interfaz. El diseñador UI se encarga de decidir el recorrido del usuario, mientras que el UX se centra en los procesos y acciones desencadenantes del uso.

Procedimientos

  • Identificar el número IP de nuestra PC: Se menciona el procedimiento de utilizar el comando "ipconfig" en la línea de comandos para obtener la dirección IP de nuestro equipo.
  • Comprobar la conectividad del servidor: Se menciona el procedimiento a utilizar con el comando "ping" seguido de la dirección IP del servidor para verificar la conectividad.
  • Verificar el recorrido de servidores: Se menciona el procedimiento de utilizar el comando "tracert" seguido de la dirección de un sitio web para ver el recorrido de los servidores hasta llegar a ese sitio.

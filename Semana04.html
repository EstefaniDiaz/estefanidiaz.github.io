¿Que temas aprendió?

Durante el estudio de JavaScript, se exploraron su historia y funciones, comprendiendo su papel central en dotar de interactividad a las páginas web y su integración mediante etiquetas de script en HTML. Se adquirió conocimiento sobre comentarios en JavaScript y PHP, variables y tipos de datos. Se investigaron operadores y estructuras de control de flujo para gestionar la ejecución del programa. Se profundizó en la creación de funciones y el uso de eventos para interacción del usuario. Además, se amplió la comprensión sobre objetos y arrays en JavaScript. Este aprendizaje detallado permitió explorar diversas aplicaciones, desde páginas web interactivas hasta aplicaciones móviles híbridas y servidores de aplicaciones, resaltando la versatilidad y utilidad de JavaScript en diversos contextos de desarrollo web y móvil.

Definiciones

Estos temas nos proporcionaron una comprensión profunda de los principios fundamentales de JavaScript. Este lenguaje de programación, interpretado y orientado a objetos, es ampliamente utilizado en el desarrollo web. Gracias a su capacidad para agregar interactividad a las páginas HTML, manipular el DOM, gestionar el flujo de ejecución y definir funciones, JavaScript se ha convertido en una herramienta indispensable para crear aplicaciones web dinámicas y atractivas. Al entender conceptos clave como su integración en HTML, el uso de comentarios, la declaración de variables y tipos de datos, los operadores, las estructuras de control y las funciones, los desarrolladores pueden aprovechar todo el potencial de este lenguaje para dar vida a sus proyectos web.

JavaScript

JavaScript es un lenguaje de programación interpretado y orientado a objetos, que se utiliza para desarrollar aplicaciones web interactivas. A medida que el lenguaje evolucionó, los desarrolladores establecieron bibliotecas, marcos y prácticas de programación, lo que permitió utilizar JavaScript tanto en el lado del cliente como en el lado del servidor.
JavaScript se utiliza en conjunto con HTML, y su código puede colocarse cerca del final de la página para asegurarse de que se cargue después del HTML sobre el que debe trabajar. Aunque JavaScript fue diseñado originalmente para su uso en navegadores web, su popularidad ha llevado a su adopción en otros entornos, como entornos JavaScript del lado del servidor.

ECMAScript

ECMAScript es un estándar de lenguaje de programación que define la sintaxis y las características del lenguaje JavaScript. Fue creado para estandarizar las diferentes implementaciones de JavaScript y garantizar la interoperabilidad entre los navegadores web y otros entornos de ejecución.

Funciones de JavaScript

Las funciones en JavaScript son bloques de construcción fundamentales que realizan tareas o calculan valores. Pueden tomar argumentos y devolver valores. También pueden ser asignadas a variables y pasadas como argumentos a otras funciones, ya que en JavaScript las funciones son objetos de primera clase.

Objetos en JavaScript

Un objeto en JavaScript es una colección de propiedades, donde cada propiedad es una asociación entre un nombre (o clave) y un valor. El valor de una propiedad puede ser de cualquier tipo, incluyendo otros objetos, funciones y arreglos.

Procedimientos

  1. JavaScript: Se presentó JavaScript como un lenguaje de programación interpretado y orientado a objetos, ampliamente empleado en el desarrollo web, resaltando su capacidad para agregar interactividad a las páginas.
  2. Integración en HTML: Se detalló cómo JavaScript se integra en HTML mediante etiquetas de script, permitiendo su ejecución en los navegadores.
  3. Comentarios en JavaScript: Se discutieron los comentarios en JavaScript, esenciales para añadir notas y mejorar la legibilidad del código, cubriendo tanto los comentarios de una línea como los de varias líneas.
  4. Variables y Tipos de Datos: Se profundizó en los conceptos de variables y tipos de datos en JavaScript, subrayando la declaración con "var" o "let" y la diversidad de tipos de datos compatibles.
  5. Operadores: Los operadores en JavaScript, incluyendo operaciones matemáticas, comparaciones y asignaciones, fueron explicados.
  6. Estructuras de Control de Flujo: También se abordaron las estructuras de control de flujo, como las sentencias if-else, while, for y switch, mostrando cómo se utilizan para tomar decisiones y repetir acciones.
  7. Funciones: Las funciones en JavaScript, que permiten encapsular bloques de código reutilizables y recibir parámetros y devolver valores, fueron analizadas.
  8. Eventos: Se hizo hincapié en la importancia de los eventos en JavaScript para responder a las acciones del usuario o cambios en la página.
  9. Objetos: Los objetos, descritos como colecciones de propiedades y métodos, así como las diferentes formas de crearlos, fueron explicados.
  10. Arrays: También se exploraron los arrays como estructuras de datos que almacenan múltiples valores en una sola variable.
  11. Aplicaciones de JavaScript: Finalmente, se señalaron las diversas aplicaciones que pueden crearse con JavaScript, desde páginas web interactivas hasta aplicaciones web, aplicaciones móviles híbridas y servidores de aplicaciones.

Resultados

  1. A partir de la respuesta HTML del servidor, se construyó el Document Object Model (DOM), que sirvió para representar la estructura y el contenido de la página web.
  2. Se desarrolló el CSS Object Model (CSSOM) para representar los estilos CSS empleados en la página. Esto posibilitó la aplicación de estilos específicos a cada elemento del DOM.
  3. Se combinaron el DOM y el CSSOM para formar el Render Tree, que mostraba los elementos visibles en la página junto con sus estilos aplicados. Esta fusión permitió determinar qué elementos serían mostrados en la pantalla y cómo se verían.
  4. Se llevó a cabo el cálculo del layout, en el cual se determinaron la posición y el tamaño exactos de cada elemento en la página. Esto implicó establecer las dimensiones y la ubicación de los elementos en relación con otros elementos y el contenedor principal.

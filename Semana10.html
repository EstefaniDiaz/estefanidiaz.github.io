¿Que temas aprendió?

En el proceso de desarrollo en React, obtuve conocimientos fundamentales al emplear el hook useState, lo que me permitió incorporar variables de estado en mis componentes para lograr la persistencia del estado entre renderizaciones y la actualización dinámica de la interfaz de usuario. Además, exploré el uso de JSX, una extensión de JavaScript utilizada en React para organizar la interfaz mediante componentes reutilizables. Comprendí el procedimiento de renderizado en React, utilizando ReactDOM.render() para presentar elementos HTML en la aplicación. También adquirí habilidades en la gestión de la transferencia de datos entre componentes, ya sea en una dirección (de padre a hijo o viceversa) o entre componentes anidados. La recomendación de estructurar los componentes en archivos separados se entendió como una medida para mejorar la reutilización y el mantenimiento del código. Finalmente, implementé la biblioteca Axios para realizar solicitudes HTTP y consumir datos de la API de Rick and Morty, demostrando así una integración efectiva de APIs externas en mis proyectos de React.

Definiciones

UseState

Un hook de React que permite agregar variables de estado a componentes funcionales, facilitando la gestión de datos que pueden cambiar a lo largo del ciclo de vida del componente.

JSX

Una extensión de JavaScript utilizada en React para escribir de manera más concisa y legible la estructura de la interfaz de usuario, combinando HTML y JavaScript.

Renderizado en React

El proceso por el cual React actualiza la interfaz de usuario para reflejar los cambios en el estado de la aplicación, utilizando el método ReactDOM.render().

Componentes en React

Unidades modulares y reutilizables que permiten organizar la lógica y la interfaz de usuario de una aplicación en bloques independientes.

Axios

Una biblioteca de JavaScript utilizada para realizar solicitudes HTTP, comúnmente empleada en el consumo de APIs.

API

Es una interfaz de procesamiento de aplicaciones entre un servidor web y un navegador web, para el ejemplo se uso una API en especifico, de la serie de televisión "Rick and Morty".

Procedimientos

  1. Uso de useState en un Componente: Se diseñó un componente empleando el hook useState con el propósito de controlar y exhibir un contador que se incrementa al hacer clic en un botón.
  2. Formulario con useState: Se creó un formulario que hace uso de useState para recopilar y visualizar la información ingresada por el usuario, ilustrando la capacidad de gestionar datos en tiempo real.
  3. Integración de Axios para Consumir API: Se empleó la biblioteca Axios para efectuar solicitudes a la API de Rick and Morty, permitiendo la presentación dinámica de detalles de personajes en función de la respuesta obtenida de la API.

Resultados

  1. Implementación de Contador con UseState: Se logró instaurar un contador interactivo que hace uso de useState para gestionar su estado y se actualiza visualmente en tiempo real.
  2. Desarrollo de Formulario Interactivo: Se creó un formulario interactivo que emplea useState para almacenar y presentar los datos introducidos por el usuario.
  3. Éxito en el Consumo de API: Se consiguió consumir la API de Rick and Morty, presentando información dinámica de personajes al activar un botón.
  4. Estructura Modular y Reutilizable: Se estructuraron los componentes en archivos separados, adhiriéndose a las prácticas recomendadas para mejorar la reutilización y la mantenibilidad del código.
  5. Aplicación de Principios Fundamentales de React: Se pusieron en práctica los conceptos clave de React, tales como el uso de JSX, el renderizado, la transferencia de datos entre componentes y la gestión de estado con useState.
  6. Creación de Componentes Visuales Atractivos y Funcionales: Se elaboraron componentes visuales atractivos y operativos, evidenciando una comprensión sólida de los principios de desarrollo en React.

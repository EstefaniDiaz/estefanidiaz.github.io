¿Que temas aprendió?

Durante el aprendizaje, se exploró el uso del elemento canvas de HTML5 en conjunto con JavaScript para crear gráficos y animaciones en páginas web. Se presentaron ejemplos de código para dibujar formas simples como líneas, rectángulos y círculos, así como para crear animaciones mediante el método setInterval(). También se abordó la aplicación del canvas en la creación de juegos básicos.

Definiciones

Durante esta semana, se adquirieron conocimientos sobre el uso del elemento canvas en HTML5, comprendiendo el contexto de dibujo, las coordenadas y la creación de animaciones en canvas mediante JavaScript. Estos conceptos permiten aprovechar al máximo las capacidades de HTML5 y JavaScript para ofrecer a los usuarios una experiencia interactiva y envolvente.

Canvas

Es un elemento de la interfaz de usuario que permite dibujar gráficos y animaciones en un lienzo HTML utilizando JavaScript. El lienzo se crea mediante el elemento canvas en HTML y se puede acceder y manipular a través del objeto CanvasRenderingContext2D en JavaScript. El uso del Canvas en el desarrollo de aplicaciones web y JavaScript ofrece varias ventajas, como:

  • Gráficos de Alta Calidad: El Canvas permite dibujar gráficos vectoriales y de mapa de bits con una resolución alta y una calidad de imagen superior.
  • Interactividad: Se pueden agregar eventos de interacción, como clics y movimientos del mouse, para permitir la interacción del usuario con los gráficos y animaciones.
  • Animaciones Fluidas: El Canvas es ideal para crear animaciones suaves y fluidas, ya que permite actualizar y redibujar rápidamente los elementos en el lienzo.
  • Compatibilidad con Dispositivos Móviles: El Canvas es compatible con dispositivos móviles y se puede utilizar para crear aplicaciones web y juegos móviles.
El uso del Canvas se ha vuelto cada vez más popular debido a su versatilidad y capacidad para crear experiencias interactivas y visualmente atractivas para los usuarios.

Contexto de Dibujo

Es la capacidad de dibujar gráficos en un lienzo HTML utilizando la API de Canvas y el contexto 2D. El lienzo es un elemento HTML que actúa como un área de dibujo en la que se pueden crear y manipular formas, líneas, imágenes y otros elementos gráficos.
El contexto de dibujo 2D proporciona una variedad de funciones y métodos para dibujar y estilizar elementos gráficos en el lienzo, como fillRect(), strokeRect(), fillText(), drawImage(), entre otros. Estos métodos permiten crear formas complejas y aplicar estilos como colores, gradientes, sombras y patrones a los elementos dibujados. Este elemento es especialmente útil para crear gráficos interactivos, juegos, visualizaciones de datos y otras aplicaciones que requieren manipulación y representación de elementos gráficos en tiempo real.

Coordenadas

Las coordenadas en canvas se utilizan para especificar la posición de los elementos gráficos en el lienzo. El sistema de coordenadas en canvas es similar al de un plano cartesiano, donde el punto (0, 0) se encuentra en la esquina superior izquierda del lienzo y los valores positivos de x y y se extienden hacia la derecha y hacia abajo, respectivamente.

Animación

Es la creación de gráficos y animaciones dinámicas en un lienzo HTML utilizando scripts de JavaScript. El elemento Canvas permite generar gráficos en pantalla, crear animaciones, manipular imágenes e incluso videos, todo ello sin necesidad de recurrir a tecnologías de terceros como Flash o los applets de Java.

Procedimientos

  1. Obtención del Contexto de Dibujo: Obtener el contexto de dibujo fue el primer paso para comenzar a dibujar en el canvas. Esto se logró mediante el método getContext() del elemento canvas, que nos proporcionó el objeto necesario para realizar las operaciones de dibujo.
  2. Dibujo de Formas Básicas: Una vez obtenido el contexto de dibujo, pudimos utilizar una variedad de métodos proporcionados por Canvas para crear formas básicas como líneas, rectángulos, círculos y arcos. Entre los métodos disponibles se encontraban strokeRect(), fillRect(), arc(), lineTo(), entre otros, que nos permitieron trazar formas precisas en el canvas.
  3. Agregado de Texto al Canvas: Además de dibujar formas, también pudimos agregar texto al canvas utilizando los métodos fillText() o strokeText(). Estos métodos nos permitieron especificar la fuente, tamaño y color del texto, ofreciendo flexibilidad en el diseño visual.
  4. Dibujo de Imágenes: Otra funcionalidad importante fue la capacidad de dibujar imágenes en el canvas utilizando el método drawImage(). Esto nos permitió especificar la imagen a dibujar, así como su posición y tamaño, lo que añadió un elemento visual adicional a nuestras creaciones.
  5. Creación de Animaciones: Para crear animaciones suaves y fluidas, aprovechamos el método setInterval() de JavaScript. Esta técnica nos permitió actualizar el contenido del canvas en intervalos regulares, creando así efectos animados dinámicos y atractivos.
  6. Aplicación de Transformaciones: Además, Canvas nos ofreció la posibilidad de aplicar transformaciones a los dibujos, como rotaciones, escalas y traslaciones. Estas transformaciones nos permitieron crear dibujos más complejos y dinámicos, añadiendo un nivel adicional de creatividad a nuestras creaciones en el canvas.

Resultados

  1. Al emplear el contexto de dibujo y la función arc() de canvas, fue posible crear una representación de un círculo.
  2. Utilizando el contexto de dibujo y las funciones rect() o fillRect() de canvas, se consiguió dibujar una figura cuadrada.

  4. Al utilizar las funciones de transformación del contexto de dibujo, como rotate() o translate(), se aplicó una transformación al cuadrado, modificando su orientación o posición en el canvas.
